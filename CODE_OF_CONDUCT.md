# Code of Conduct

## Our standard

We want participation in Material Event Radar to be respectful, factual, and welcoming. Constructive disagreement is encouraged; harassment, personal attacks, discriminatory language, threats, doxxing, and deliberate disruption are not acceptable.

Examples of positive participation include:

- focusing feedback on behavior, evidence, and user impact;
- respecting different levels of experience;
- acknowledging uncertainty and correcting mistakes;
- protecting private and security-sensitive information.

## Scope

This standard applies in repository Issues, pull requests, Discussions, reviews, and other project spaces, as well as when someone officially represents the project.

## Enforcement

Maintainers may edit or remove content and may temporarily or permanently restrict participation when behavior violates this standard. Enforcement decisions should be proportionate, documented privately, and protect the reporter where possible.

To report conduct privately, contact the repository owner through the contact options on the [huluwa2026 GitHub profile](https://github.com/huluwa2026). Do not disclose sensitive personal information in a public Issue.
